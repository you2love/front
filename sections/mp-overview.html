    微信公众号 - 概述

    微信公众号是微信生态中面向企业和个人的内容与服务平台，可通过推送消息、提供服务、网页开发等方式触达用户。

    公众号特点

    用户触达

    直接向关注用户推送图文消息，打开率高

    服务能力

    提供自定义菜单、模板消息、客服等服务能力

    网页开发

    基于JS-SDK实现拍照、支付、分享等原生能力

    开放接口

    丰富的API接口，可与自有系统深度集成

    技术架构

    层面 技术栈 说明
    服务端 Node.js / Python / Java 接收微信服务器消息，处理业务逻辑
    消息通信 XML / JSON 微信服务器与开发者服务器之间的数据格式
    网页前端 HTML/CSS/JS + JS-SDK 公众号内嵌网页，调用微信原生能力
    认证授权 OAuth 2.0 获取用户信息，实现网页授权登录
