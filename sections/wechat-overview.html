    微信小程序 - 概述

    微信小程序是一种不需要下载安装即可使用的应用，它实现了应用"触手可及"的梦想，用户扫一扫或者搜一下即可打开应用。

    小程序特点

    即用即走

    无需安装，扫码即用，用完即走

    开发成本低

    使用类Web技术栈，降低开发门槛

    生态丰富

    依托微信生态，用户基数庞大

    功能强大

    支持原生能力，接近原生体验

    技术架构

    层面 技术栈 说明
    渲染层 WebView + WXML/WXSS 负责界面渲染，类似Web页面
    逻辑层 JavaScript 处理业务逻辑，运行在JSCore中
    原生能力 Native API 提供摄像头、地理位置等原生功能
    通信层 JSBridge 连接逻辑层和渲染层
